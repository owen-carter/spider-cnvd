# spider-cnnvd
> 一个爬取cnnvd的爬虫

### How to start
+ 手动方式
    ```bash
    git clone https://github.com/owen-carter/spider-cnnvd.git
    cd spider-cnnvd
    npm install
    npm start
    ```
+ 快捷方式
    ```bash
    curl -o- https://raw.githubusercontent.com/owen-carter/spider-cnnvd/master/start.sh | bash
    ```



### About 
- 使用es6编写
- 大量使用es6的特性
- 重点学习requestretry的使用
- 支持真实的并发
- 可以存mongodb
- 可以写文件
